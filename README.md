# CASEFILES

A mobile-first detective game foundation built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in landscape orientation on a mobile device or browser dev tools.

## Features

- Immersive night office desk hub
- Four interactive objects: Monitor, Phone, Folder, Evidence Board
- Cinematic Framer Motion transitions
- Mobile landscape-first, fullscreen experience

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS 4
- Framer Motion
